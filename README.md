# Dotfiles - Avasz

To Backup:
```
./dotfiles backup
```

To Restore:
```
./dotfiles restore
```
The following are backed-up:
- .Xresources
- .vimrc
- .tmux.conf
- .bashrc
- .config/
	- i3/
	- i3blocks/
	- rofi/
	- picom
	- ranger/
	- alacritty/
- List of installed packages via `pacman -Qe`
